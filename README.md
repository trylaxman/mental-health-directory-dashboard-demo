# Mental Health Directory Dashboard Demo

A modern directory platform demo built for the mental health industry.

## Features

### Public Directory
- Search professionals
- Filter by topic, location, price, format, and accreditation
- Listing detail pages

### Trainer Portal
- Account registration
- Listing submission and management
- Subscription management
- Billing status overview

### Admin Dashboard
- Listing approval workflow
- User management
- Taxonomy and tag management
- Platform reporting

### Billing
- Stripe subscription workflow simulation
- Webhook event simulation
- Account status management

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- LocalStorage (Demo Data)
- Mock Stripe Workflows

## Purpose

This project was created as an investor/client-facing prototype to demonstrate the complete user experience prior to production implementation.

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```
